Operating system requirements:
Windows 95 or later

Installation Instructions:
1)Create a new folder on your hard drive named MICRbase
  or whatever name you prefer.  The remaining instructions
  assume you used the name MICRbase.
2)Copy all the files and folders from the CD into
  the new MICRbase folder.
3)For wedge-based products on Windows NT operating systems only, follow the additional
  instructions in the file \MICRbase\MagEpNt\Regupdt.txt
4)For wedge-based products on Windows 2K and XP operating systems only, follow the additional
  instructions in the file \MICRbase\MTKBHOOK\Install.htm
Running MICRbase:
To run MICRbase run \MICRbase\MICRbase.exe
